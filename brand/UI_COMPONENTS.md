# AuthePay UI Components

## Purpose

This document defines the official user interface components for the AuthePay platform.

All web applications, mobile applications, dashboards, merchant portals, customer portals, admin systems, and APIs must use these reusable components.

---

# Design Principles

- Clean
- Modern
- Fast
- Accessible
- Responsive
- Enterprise Grade
- Mobile First
- Consistent

---

# Buttons

Primary Button

Purpose

Main actions

Examples

- Pay Now
- Send Money
- Continue
- Save
- Submit

Style

- Primary Blue background
- White text
- Rounded corners
- Medium shadow
- Smooth hover animation

---

Secondary Button

Purpose

Alternative actions

Examples

- Cancel
- Back
- Learn More

Style

- White background
- Blue border
- Blue text

---

Danger Button

Purpose

Destructive actions

Examples

- Delete
- Remove
- Block
- Reject

Style

- Red background
- White text
